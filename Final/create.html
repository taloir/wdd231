Description:

This is the physical description of the character. It can sometimes also include descriptions of the characters abilities.

Types:

Each character has two types from the following list - Aggregate, Animated, Aquatic, Beastial, Ethereal, Exotic, Huge, Magic, Mobile, Normal, Ranged, Sacred, Tiny, Unholy, Verdant, Volatile

Def score:

This is a score from 1-30 representing how difficult the character is to injure. Basic characters usually have a Def of 7-13

Personality:

Each character has a basic personality description to help inspire roleplay. These can be single words, or even a full sentence, but they should stay brief.

Atk score:

This is a score from 1-30 representing how dangerous and aggressive the character is. Basic characters usually have an Atk of 7-13
